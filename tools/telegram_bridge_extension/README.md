# Antigravity Telegram Bridge

Extension này kết nối Telegram Bot với VS Code Agent.
Nó cho phép gửi yêu cầu từ Telegram trực tiếp vào Agent Panel và nhận phản hồi ngược lại.

## Tính năng
- Đồng bộ tin nhắn Telegram.
- Hàng đợi (Queue) khi Agent đang bận.
- Typing indicator trên Telegram.
- Periodic execution (chạy lệnh định kỳ).
